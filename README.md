# Sistema Administrativo Integrado (SAI)

## Sobre o Projeto

Sistema desenvolvido em Python para automação de processos administrativos, visando a redução de trabalho manual e mitigação de erros operacionais.

O projeto segue uma arquitetura modular, dividindo responsabilidades entre interface de usuário e processamento de dados.

## Funcionalidades Atuais

- **Autenticação:** Sistema de Login seguro.
- **Relatórios Gerenciais:** Dashboard interativo com gráficos de vendas e metas (KPIs).
- **Consolidação de Dados:** Unificação automática de múltiplas planilhas Excel/CSV.
- **Cobrança Inteligente:** Envio de e-mails personalizados com seleção de tom de voz.
- **Organização de Arquivos:** Renomeação automática de PDFs baseada em RegEx (OCR).

## Tecnologias Utilizadas

- **Linguagem:** Python 3.12+
- **Frontend:** Streamlit
- **Visualização de Dados:** Plotly & Pandas
- **Banco de Dados:** SQLite (Migração prevista para PostgreSQL)
- **Manipulação de Arquivos:** PyPDF, OpenPyXL, XlsxWriter
